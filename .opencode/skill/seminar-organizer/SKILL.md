# Seminar Organizer Skill

此技能旨在自动化组会后的文档处理工作。

## 核心功能
1. **身份识别**：读取 `/People` 目录下所有 Markdown 文件的 YAML frontmatter 中的 `name` 字段，建立“中文名 -> 文件名”的映射表。
2. **文本总结**：提取录音转录文本中的核心讨论点（特别是 FEM/Meshfree 相关技术细节）。
3. **任务分发**：识别任务分工对话，生成格式为 `- [ ] 任务描述 [[文件名]]` 的待办事项。

## 使用指南
输入组会笔记的路径，该技能将：
- 提取笔记中的原始录音文本。
- 填充『会议纪要』部分。
- 在『待办事项』部分生成任务列表。
- 使用 `[[WuJC]]` 这种不带路径前缀的格式进行人名引用。
