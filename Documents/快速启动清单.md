# 快速启动清单（First 90 Days）
请按以下步骤有序启动你的研究生生涯：

### **第1-2周：安顿与熟悉**
- [ ] 阅读本指南。
- [ ] 开通所有账户（Github）。
- [ ] 与课题组每位成员进行一次简短交流（>15分钟）。
- [ ] 配置好个人电脑的基础开发环境。

### **第3-4周：技能筑基**
- [ ] 完成Git基础学习，并在GitHub上创建第一个仓库。
- [ ] 使用Julia完成一个简单有限差分或有限元程序（例如：一维热传导）。

### **第5-8周：进入领域**
- [ ] 精读导师指定的3-5篇核心文献，并撰写一份2页的综述摘要。
- [ ] 在组会上做一次15分钟的文献分享。
- [ ] 与导师讨论，初步确定2-3个可能的课题方向。

### **第9-12周：课题启动**
- [ ] 针对一个选题，完成一份初步的《研究计划书》（约3页，含研究背景、目标、初步技术路线）。
- [ ] 在导师指导下，细化第一个研究季度的具体任务。
- [ ] **目标：在第三个月末，形成明确、可执行的课题启动方案。**
